# 📧 AI Email Assistant

مساعد ذكي للبريد الإلكتروني مبني بـ Python و Claude AI. يساعدك على كتابة، ترجمة، وتلخيص الإيميلات بشكل احترافي.

## ✨ المميزات

- ✍️ **كتابة إيميلات احترافية**: من نقاط بسيطة إلى إيميل كامل بنبرة وأسلوب مهني
- 🔄 **ترجمة ذكية**: ترجمة الإيميلات بين اللغات مع الحفاظ على الطابع المهني
- 📝 **تلخيص الإيميلات الطويلة**: استخراج النقاط الأساسية والمطلوب منك
- 💬 **اقتراح ردود مناسبة**: رد احترافي جاهز (قبول، رفض مهذب، طلب توضيح، تأجيل)
- 🌐 **دعم متعدد اللغات**: العربية والإنجليزية والفرنسية
- 💼 **مناسب للعمل المهني**: مصمم للبيئات الرسمية والمؤسسية

## 🚀 التثبيت والإعداد

### المتطلبات
```bash
pip install anthropic
```

### الحصول على API Key
1. اذهبي إلى [console.anthropic.com](https://console.anthropic.com)
2. أنشئي حساباً أو سجّلي دخول
3. أنشئي مفتاح API جديد من Settings → API Keys
4. احفظي المفتاح في مكان آمن

### إعداد متغير البيئة

**على Windows (PowerShell):**
```powershell
$env:ANTHROPIC_API_KEY="your-api-key-here"
```

**على Linux/Mac:**
```bash
export ANTHROPIC_API_KEY="your-api-key-here"
```

## 📖 الاستخدام

### التشغيل التفاعلي
```bash
python email_assistant.py
```

سيظهر لكِ قائمة بالخيارات:
```
===========================================
    📧 مساعد البريد الإلكتروني بالذكاء الاصطناعي
===========================================

1️⃣  كتابة إيميل جديد
2️⃣  ترجمة إيميل
3️⃣  تلخيص إيميل
4️⃣  اقتراح رد
5️⃣  خروج
```

### الاستخدام البرمجي

```python
from email_assistant import EmailAssistant

assistant = EmailAssistant()

# كتابة إيميل
email = assistant.compose_email(
    points="طلب إجازة يوم الخميس القادم لظروف شخصية",
    tone="رسمي",
    language="العربية"
)
print(email)

# ترجمة إيميل
translated = assistant.translate_email(
    email_text="مرحباً، أرغب بتحديد موعد اجتماع",
    target_language="English"
)
print(translated)

# تلخيص إيميل
summary = assistant.summarize_email(long_email_text)
print(summary)

# اقتراح رد
reply = assistant.suggest_reply(
    email_text="original email here",
    reply_type="قبول"
)
print(reply)
```

## 💡 أمثلة عملية

### مثال 1: كتابة إيميل طلب إجازة
**الإدخال:**
```
- طلب إجازة يوم الخميس 25 أبريل
- لظروف شخصية عائلية
- سأكون متاحة للرد على الإيميلات المهمة
```

**الإخراج:**
```
Subject: طلب إجازة ليوم 25 أبريل

السلام عليكم،

أتشرف بتقديم طلب إجازة ليوم الخميس الموافق 25 أبريل، وذلك 
لظروف شخصية عائلية طارئة...
```

### مثال 2: تلخيص إيميل طويل
**الإخراج:**
```
📧 الموضوع الرئيسي: تحديث جدول الاجتماعات الأسبوعية

🎯 النقاط الأساسية:
• تأجيل اجتماع الإثنين إلى الثلاثاء
• إضافة اجتماع طارئ يوم الأربعاء
• تغيير موقع الاجتماع إلى القاعة B

⚡ المطلوب منك: تأكيد حضورك قبل يوم الأحد
⏰ الإلحاح: متوسط
```

## 🔬 كيف يعمل؟

يستخدم المشروع **Claude API** من Anthropic، أحد أقوى نماذج الذكاء الاصطناعي في العالم. كل وظيفة تعتمد على:

1. **Prompt Engineering** دقيق لتوجيه النموذج
2. **سياق مهني** واضح لضمان جودة المخرجات  
3. **تنسيق منظم** للحصول على نتائج متناسقة
4. **معالجة الأخطاء** للتعامل مع مشاكل الشبكة أو API

## 🛡️ الأمان والخصوصية

- ✅ API key يُقرأ من متغيرات البيئة، لا يُخزَّن في الكود
- ✅ لا يتم حفظ الإيميلات محلياً
- ✅ كل طلب منفصل ولا يُربط بطلبات سابقة
- ⚠️ **تنويه**: لا تستخدمي الأداة لإيميلات تحتوي معلومات حساسة جداً
